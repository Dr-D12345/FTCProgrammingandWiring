# Java Data Type

- ## Int 

  - Int accept an integer value
- ## Double

  - Double accept a decimal value
- ## Boolean

  - Boolean accept true / false
- ## Char

  - Char accept a single character
- ## String

  - String accept a string that contain numbers and characters

# Declare a Variable

```java
int myNumber;
myNumber = 5;
```

OR

```java
int myNumber = 5;
```
